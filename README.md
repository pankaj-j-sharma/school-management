Dashboard

    monthly income with graph
    total income
    monthly student registration

MyProfile
    Edit/Update

Classroom
    edit/update/delete/list classroom ( with student count )

Grade

Subject
    edit/update/delete/list Subject

Teacher
    edit/update/delete/list Teacher

Subject Routing

Attendance

Exam
    Create Exam
    Exam Timetable
    Student Exam Marks

PoolMoney
    Issue pool money

Friends
    add/view friends

Events
    create/view events
    notify events
